As long as the above status says "Connected", you should be able to use any of the available control methods simultaneously:

  • The buttons below simply send whatever they say on them. The layout matches a standard Switch controller, roughly.
  • There is a drop-down list that says "Copy controller inputs from: None". If you have a real controller/gamepad connected to your computer (any kind), pressing any button on it should make it show up in the list. Select it in the list and any presses on the controller will be copied. Not all controllers may work out of the box, and there is no remapping support; but you can just open the source of this webpage and add your own mapping by hand for non-standard controllers. üëç
  • Finally, the bottom of the page has a blank textfield with a "Run script" button underneath. Pressing the button will queue the contents of the text field as button presses. Script syntax can be found in the project README file!
